高中文言文实词

爱：

  1. 喜爱.《师说》：“爱其子,择师而教之。”
  2. 爱护.《陈涉世家》：“吴广素爱人,士卒多为用者。”
  3. 怜惜;同情.《左传·僖公二十二年》：“若爱重伤,则如勿伤。”
  4. 亲爱的;心爱的.《柳毅传》：“见大王爱其女牧羊于野。”
  5. 吝啬.《齐桓晋文之事》：“吾何爱一丰?。”
  6. 爱好.《阿房宫赋》：“秦爱纷奢,人亦念共家。”

鄙：

  1. 边远的地方。《为学》：“蜀之鄙有二僧，其一贫，其一富。”
  2. 鄙陋，鄙俗。《曹刿论战》：“肉食者鄙，未能远谋。”《廉颇蔺相如列传》
  3. 轻贱。《孔雀东南飞》：“人贱物亦鄙，不足迎后人。”
  4. 鄙薄，轻视。《训俭示康》：“孔子鄙其其小器。”
  5. 谦词。《滕王阁序》：“敢竭鄙诚。”

安：

  1. 安宁;安全.《茅屋为秋风所破歌》：“风雨不动安山。”
  2. 安逸.《生于忧患,死于安乐》：“然后知生于忧患,而死于安乐也。”
  3. 养;奉养.《曹刿论战》：“衣食所安,弗敢专也,必以分人。”
  4. 安置;安放.《失街亭》：“离山十里,有王平安营。”
  5. 安心;习惯于.《柳敬亭传》：“敬亭亦无所不安。”
  6. 表处所.哪里;什么地方.《鸿门宴》：“沛公安在?”

被：

  1. 被子。
  2. 覆盖。《促织》：“成归,闻妻言,如被冰雪。”
  3. 遭受;蒙受。《荆轲刺秦王》：“秦王复击轲,被八创。”
  4. 表被动。《屈原列传》：“信而见疑,忠而被谤。”
  5. 通“披”（pi）,穿在身上或披在身上。《陈涉世家》：“将军身被坚执锐,伐无道。”

薄：

  1. 草木丛生的地方。《涉江》：“露申辛夷，死林薄兮。”
  2. 微少；微薄；浅薄。《促织》：“不终岁，薄产累尽。”
  3. 厚度小，与“厚”相对。《白雪歌送武判官归京》：“狐裘不暖锦衾薄。”
  4. 鄙薄；轻视。《出师表》：“不宜妄自菲薄，引喻失义。”
  5. 通“迫”，迫近，靠近。《涉江》：“腥臊并御，芳不得薄兮。”
  6. 通“迫”，附着，混杂。《狱中杂记》：“与饮食之所相薄。”
  7. 减轻。晁错《论贵粟疏》：“薄赋敛，广畜积。”

拜：

  1. 古代表示恭敬的礼节。两手合于掌于胸前，头低到挨着手，后世指下跪叩头。《陈情表》：“臣不胜犬马怖惧之情，谨拜表以闻。”《鸿门宴》：“哙拜谢，起，立而饮之。”
  2. 授给官职。《廉颇蔺相如列传》：“又以相如功大，拜为上卿。”
  3. 接受任命。《指南录后序》：“于是辞相印不拜。”

兵：

  1. 兵器;武器。《殽之战》：“束载、厉兵、秣马矣。”《过秦论》：“斩木为兵,揭竿为旗。”
  2. 军队;士兵。《谋攻》：“不战而屈人之兵。”
  3. 战争;军事。《论积贮疏》：“兵旱相乘,天下大屈。”《教战守策》：“昔者先王知兵之不可去也。”
  4. 战略。《谋攻》：“战上兵伐谋。”

达：

  1. 通畅。《送东阳马生序》：“撰长书以为贽，辞甚畅达。”
  2. 到；到达。《愚公移山》：“指通豫南，达达于汉阴。”
  3. 通晓，明白。韩愈《张中丞传后叙》：“其亦不达于理矣。”
  4. 豁达；心怀宽阔。《汉书·高帝纪》：“高祖不修文学，而性明达。”
  5. 显达；显贵。《出师表》：“苟全性命于乱世，不求闻达于诸候。”
  6. 有道德、学有问的人。《送东阳马生序》：“尝趋百里外从乡之先达执经叩问。”
  7. 传达；送到。《柳毅传》：“时有宦人密侍君者，君以书授之，命达宫中。”

本：

  1. 草木的根。《谏太宗十思疏》：“臣闻求木之长者,必固其根本。”
  2. 根本;基础。《齐桓晋文之事》：“王欲行之,刎盍反其本矣。”
  3. 根本;基础。《齐桓晋文之事》：“王欲行之,刎盍反其本矣。”
  4. 本源;根源。《原毁》：“为是者有本有原,怠与忌之谓也。”
  5. 依照;依据。《图画》：“虽理想派之作,亦先有本,乃增损而润色之。”
  6. 堆原,考察。《伶官传序》：“抑本其成败之迹,而皆自于人欤?”
  7. 本来的,原来的。《孟子·鱼我所欲也》：“此之谓失其本心。”
  8. 这里的;自己的。《采草药》：“用芽者自从本说。”
  9. 臣下给皇帝的奏章或书信。《狱中杂记》：“是无难,别具本章。”
  10. 书籍的一册。《活板》：“若止印三二本,未为简易。”

病：

  1. 病加重。《论语·述而》：“子疾病，子路请祷。”
  2. 泛指疾病。《察今》：“病万变，药亦万变。”
  3. 重病。《扁鹊见蔡桓公》：“君之病在肌肤。”
  4. 生病。《廉颇蔺相如列传》：“相如每朝时，常称病。
  5. 弊病；毛病；缺点。《原毁》：“不如舜，不如周公，吾之病也。”
  6. 羞辱；伤害。《答韦中立论师道书》：“非独见病，亦以病吾子。”
  7. 担忧；忧虑。《论语·卫灵公》：“君子病无能焉，不病人之不己知也。”

表：

  1. 外衣。《庄子·让王》：“子贡乘大马，中绀而表素。”
  2. 外表；外面。《赤壁之战》：“江表英豪咸归附之。”
  3. 上面。《复庵记》：“绵上之山出没于云烟之表。”
  4. 标记；标志。《察今》：“循表而夜涉。”
  5. 臣下给皇帝的奏章，如《出师表》、《陈情表》。

操：

  1. 拿着；携带。《愚公移山》：“操蛇之神闻之，惧其不已也，告之于帝。”
  2. 掌握；控制。《韩非子·定法》：“操杀生之柄。”
  3. 持用；采取。《答司马谏议书》：“而议事每不合，所操之术多异故也。”
  4. 从事。《促织》：“邑有成名者，操童子业，久不售。”
  5. 节操；品行。《史记·张汤传》：“汤之客田甲，虽贾人，有贤操。”
  6. 琴曲。《列子·伯牙与钟子期》：“初为霖雨之操，更造崩山之音。”
  7. 操守；品行。《楚辞·七谏·沉江》：“正臣端其操兮，反离谤而见攘。”

陈：

  1. 古指台阶，及台阶下的地方。《冯谖客孟尝君》：“狗马实外厩，美人充下陈。”
  2. 陈列；摆列。《荆轲刺秦王》：“诸郎中执兵，皆陈殿下。”
  3. 陈述；陈说。《荆轲剌秦王》：“恐惧不敢自陈。”
  4. 旧；旧的。《兰亭集序》：“向之所欣，俯仰之间，已为陈迹。”
  5. 交战时的战斗队列。《史记·李将军列传》：“广行无部伍行陈。”
  6. 列阵；布阵。《子鱼论战》：“既陈而后击之。”

冲：

  1. 向上冲。《廉颇蔺相如列传》：“相如因持璧却立，倚柱，怒发上冲冠。”
  2. 虚；空虚。《老子》：“大盈若冲，其用不穷。”
  3. 谦虚。《谏太宗十思疏》：“念高危则思谦冲而自牧。”
  4. 交通要道。《失街亭》：“今令汝接应街亭，当阳平关冲要道路，总守汉中咽喉。”
  5. 快速向前冲。《雁荡山》：“原其理，当是为谷中大水冲激，沙土尽去。”
  6. 碰撞；冲击。《黔之驴》：“稍近，益狎，荡倚冲冒。”
  7. 侵袭；侵犯。《教战守策》：“其筋骸之所冲犯。”

尝：

  1. 吃；品尝。《察今》：“尝一脟肉，而知一镬之味，一鼎之调。”
  2. 尝试。《齐桓晋文之事》：“我虽不敏，请尝试之。”
  3. 经历；遇到。《庖丁解牛》：“技经肯綮之未尝，而况大軱乎！”
  4. 曾；曾经。《劝学》：“吾尝终日而思矣，不如须臾之所学也。”

诚：

  1. 真诚；诚实。《谏太宗十思疏》：“盖在殷忧必竭诚以待下。”
  2. 诚心。《陈情表》：“愿陛下矜悯愚诚。”
  3. 真实；真实的。《史记·扁鹊列传》：“子以吾言为不诚，试入诊太子。”
  4. 实在；确实。《邹忌讽齐王纳谏》：“臣诚知不如徐公美。”
  5. 果真；如果确实。《屈原列传》：“楚诚能绝齐，秦愿献商、於之地六百里。”

多：

  1. 数量大。《登泰山记》：“石苍黑色，多平方，少圜。”
  2. 多多地；大量地。《岳阳楼记》：“迁客骚人，多会于此。”
  3. 称赞；赞美。《五蠹》：“故传天下而不足多也。”
  4. 只；仅仅。《论语·子张》：“多见其不知量也。”

绝：

  1. 断；断绝。《共工头触不周山》：“天柱折，地维绝。”
  2. 隔绝；隔断。《三峡》：“至于夏水襄陵，沿溯阻绝。”
  3. 绝交；断绝往来。《屈原列传》：“楚怀王贪而信张仪，遂绝齐。”
  4. 停止；止歇。《与朱元思书》：“蝉则千转不穷，猿则百叫无绝。”
  5. 绝命；死。《孔雀东南飞》：“我命绝今日，魂去尸长留。”
  6. 横渡。《劝学》：“非能水也，而绝江河。”
  7. 旧诗体裁之一。王实甫《西厢记》：“我且高吟一绝，看他则甚。”
  8. 到极点的；不能超越的。《与朱元思书》：“奇山异水，天下独绝。”
  9. 《核舟记》：“佛印绝类弥勒。”《游黄山记》：“众壑纵横，真黄山绝胜处。”
  10. 绝对；完全。《促织》：“而心目力俱穷，绝无踪响。”

审：

  1. 详细；周密。《促织》：“审视，巨身修尾，青项金翅。”
  2. 仔细观察；考察。《察今》：“故审堂下之阴，而知日月之行。”
  3. 清楚了解。《归去来兮辞》：“倚南窗以寄傲，审容膝之易安。”
  4. 审问；审判。《狱中杂记》：“馀经秋审，皆减等发配。”
  5. 确实；的确。《梅花岭记》：“先生在兵间，审知故扬州阁部史公果死耶，抑未死耶。”
  6. 谨慎；慎重。《韩非子·存韩》：“兵者凶器也，不可不用也。

善：

  1. 好；善良。《〈论语〉六则》：“择其善者而从之，其不善者而改之。”
  2. 喜欢；羡慕。《归去来兮辞》：“善万物之得时，感吾生之行休。”《陌上桑》：“罗敷善蚕桑，采桑城南隅。”
  3. 表示赞许。同意的应答之词。《邹忌讽齐王纳谏》：“王曰：‘善。’乃下令。”
  4. 好的行为；长处。《墨池记》：“惟王君之心，岂爱人之善，虽一能为以废，而因为及其迹邪？”《原毁》：“一善易能也。”
  5. 亲善；友好。《鸿门宴》：“楚左尹项伯者，项羽季父也，素善留侯张良。”
  6. 善于；擅长。《劝学》：“君子生非异也，善假于物也。”《卖油翁》：“陈康肃公尧咨善射。”
  7. 好好地。《荆轲刺秦王》：“愿得将军之首以献秦，秦王必喜而善见臣。”
  8. 熟悉。《林黛玉进贾府》：“我看着面善，心里就算是旧相识。”
  9. 通“缮”。修治；整理。《庖丁解牛》：“善刀而藏之。”

并：

  1. 合。《六国论》：“以事秦之心礼天下之奇才,并力西向。”
  2. 兼并;吞并。《过秦论》：“囊括四海之意,并吞八荒之心。”
  3. 并列;挨在一起。《与妻书》：“吾与(汝)并肩携手,低低切切。”
  4. 一并;一起。《涉江》：“腥臊并御,芳不得薄兮。”
  5. 连。《芋老人传》：“固已贤夫并老人而芋视之者。”
  6. 并且。《〈黄花冈七十二一烈士事略〉序》：“予为斯序，既痛逝者，并以为国人之读兹编者勖。

施

  1. 实施；实行。《齐桓晋文之事》：“今王发政施仁，使天下仕者皆欲立于王之朝。”
    1. 施展；使用。《促织》：“靡计不施，迄无济。”
    2. 施加；推与。《论语·颜渊》：“己所不欲，勿施于人。”
    3. 搽抹。《林黛玉进贾府》：“唇若施脂。”
    4. 施舍；给予。《孔雀东南飞》：“留待作遗施，于今无会因。”
    5. 恩惠；好处。《殽之战》：“未报秦施而伐其师。”
    6. 设置；安放。《口技》：“会宾客大宴，于厅事之东北角，施八尺屏障。”
    7. 通“弛”。放松；解除。《后汉书·光武帝纪》：“遣骠骑大将军杜茂将众郡施刑屯北边。”

    8. 邪；斜。《韩非子·诡使》：“谄施、顺意、从欲以危世者，近习。”
    9. 斜行。《孟子·离娄》：“蚤起，施从良人之所之。”
      (yi)
    10. 延伸；延续。《叶公好龙》：“于是天龙闻而下之，窥头于牖，施尾于堂。”

悲：

  1. 悲哀;伤心。《柳毅传》：“悲泗淋漓,诚怛人心。”《岳阳楼记》：“不以物喜,不以己悲。”
    可悲的。《察今》：“以为治岂不悲哉。”
  2. 怜悯,同情。《捕蛇者说》：“余悲之,且曰‘若毒之乎?’。”《指南录后序》：“使来者读之,悲予志焉。”
  3. 顾念;怀念。《汉书·高帝纪》：“游子悲故乡。”
  4. 悲伤。《游褒禅山记》：“又以悲夫古书之不存。”

废：

  1. 舍弃；停止。《齐桓晋之事》：“然则废衅钟与?”《陈情表》：“臣侍汤药，未曾废离。”
  2. 废黜；罢黜。韩愈《柳子厚墓志铭》：“谓功业可立就，故坐废退。”
  3. 衰败。《岳阳楼记》：“政通人和，百废待兴。”
  4. 秃败；倒塌。《扬州慢》：“废池乔木，犹厌言兵。”
  5. 负伤；残废。《荆轲刺秦王》：“荆轲，乃引其匕首提秦王。”
  6. 浪费。《乐羊子妻》：“今若断斯织也，则捐失成功，稽废时日。”
  7. 失望；颓废。《原君》：“其欲得天下之心，有不废然摧泪者乎。”

就：

  1. 接近；靠近；趋向。《劝学》：“木受绳则直，金就砺则利。”《赤壁之战》：“邂逅不如意，便还就孤。”
  2. 上；上登。《信陵君窃符救赵》：“乃谢客就车。”
  3. 就任；就职。《陈情表》：“臣具以表闻，辞职不就。”
  4. 参加；参与。《芋老人传》：“知从郡城就童子试归。”
  5. 承受；接受。《殽之战》：“使归就戮于秦。”
  6. 择取；效法。《原君》：“去其不如舜者，就其如舜者。”
  7. 成就；成功。《乐羊子妻》：“当日知其所亡以就懿德。”《荆轲刺秦王》：“轲自知事不就，倚柱而笑。”
  8. 看；观赏。《过故人庄》：“等到重阳日，还来就菊花。”
  9. 向；从；跟。《狱中杂记》：“余尝就老胥而求焉。”
  10. 就着；根据。《芋老人传》：“然就其不忘一芋，固已贤夫并老人而芋视之者。”
  11. 即便；即使。《三国志·蜀书·法正传》：“就复东行，必不倾危矣。”
  12. 即，便。《红楼梦》：“至院外，就有跟贾政的小厮上来抱住。”

却：

  1. 退；后退。《廉颇蔺相如列传》：“相如因持璧却立。”
  2. 打退；击退。《六国论》：“后秦击赵者再，李牧连却之。”
  3. 推却；推辞；拒绝。李斯《谏逐客书》：“王者不却众庶。”
  4. 去；去掉。聂夷中《伤田家》：“医得眼前疮，剜却心头肉。”
  5. 回头。《孔雀东南飞》：“却与小姑别，泪落连珠子。”
  6. 再；重；又。《闻官军收复河南河北》：“却看妻子愁何在。”
  7. 通“恰”，刚刚。《长亭送别》：“却告了相思回避，破题儿又早别离。”

疏：

  1. 疏导；疏通。《答韦中立论师道书》：“疏之欲其通。”
  2. 疏远。《屈原列传》：“王怒而疏屈平。”
  3. 分；分散。《史记·黥布列传》：“上裂地而王之，疏爵而贵之。”
  4. 稀疏。《与朱元思书》：“疏条交映，有时见日。”
  5. 少；稀少。《杜十娘怒沉百宝箱》：“妾久疏谈笑。”
  6. 粗疏。《史记·范雎列传》：“其于计疏矣。”
  7. 粗糙的米。韩愈《山石》：“铺床拂席置羹饭，疏粝亦足饱我饥。”
  8. 分条陈述。《滕王阁序》：“敢竭鄙怀，恭疏短引。”
  9. 给皇帝的奏议。如《论积贮疏》、《谏太宗十思疏》
  10. 古书注释的一种。不仅注释古书原文，对前人的注释也加以申发说明。
  11. 僧道拜忏时焚化的祝告文。《京本通俗小说·志诚张主管》：“今日是员外生辰，小道送疏在此。”

见：

  1. 看；看到。《齐桓晋文之事》：“见牛未见羊也。”
  2. 遇见；碰见。《回乡偶书》：“儿童相见不相识。”
  3. 见面；会面。《琵琶行》：“移船相近邀相见。”
  4. 拜见；谒见。《邹忌讽齐王纳谏》：“于是入朝见威王。”
  5. 召见；接见。《廉颇蔺相如列传》：“秦王坐章台见相如。”
  6. 知道；懂得。《示儿》：“乃翁见事可怜迟。”
  7. 听见；听到。《林黛玉进贾府》：“总不许见哭声。”
  8. 见解；见识。《晋书·王浑传》：“敢陈愚见。”
  9. 用在动词前，表被动，译为“被”。《屈原列传》：“信而见疑，忠而被谤。”
  10. 在动词前，表示说话人自己，可译为“我”。《孔雀东南飞》：“君既若见录，不久望君来。”
    Xian
  11. 同“现”，出现；显现。《毛遂自荐》：“锥之处囊中，其未立见。”
  12. 使……拜见。《荷蓧丈人》：“止子路宿，杀鸡为黍而食之，见其二子焉。”

得：

  1. 获得；得到；取得。与“失”相对。《鱼我所欲也》：“一箪食，一豆羹，得之则生，弗得则死。”《垓下之战》：“汉皆已得楚乎？”
  2. 贪求；贪得。《论语·季氏》：“戒之在得。”
  3. 心得；收获。《游褒禅山记》：“古人之观于天地，……往往有得。”
  4. 得意；满足。《兰亭集序》：“当其欣于所遇，暂得于己，快然自足。”
  5. 领会；理解。《朝三暮四》：“宋有狙公者，爱狙，养之成群，能解狙之意，狙亦得公之心。”
  6. 合适；恰当。《六国论》：“此言得之。”
  7. 实现；达到。《信陵君窍符救赵》：“自王以下，欲求报其父仇，莫能得。”《谏太宗十思疏》：“既得志则纵情以傲物。”
  8. 能；能够。《垓下之战》：“项王自度不得脱。”《送东阳马生序》：“以是人多以书假余，余因得遍观群书。”
  9. 应该；应当。《鸿门宴》：“君为我呼入，吾得兄事之。”
    10.表示完成或补充说明结果。《琵琶行》：“十三学得琵琶成，名属教坊第一部。”《永遇乐·京口北固亭怀古》：“元嘉草草，封狼居胥，赢得仓皇北顾。”
  10. 必须。《红楼梦》：“这件事还得你去才弄的明白。”

笃：

  1. 忠诚；厚道。《答韦中立论师道书》：“仆道不笃，业甚浅近。”
  2. 坚定。《礼记·中庸》：“明辨之，笃行之。”
  3. （病）重。《陈情表》：“臣欲奉诏奔驰，则刘病日笃。”
  4. 很；非常。《南史·文学传》：“盖由时主儒雅，笃好文章。”

分：

fēn

  1. 剖开；分开。《谋攻》：“倍则分之。”?滕王阁序》：“星分翼轸，地接衡庐。”
  2. 半；一半。《三峡》：“自非亭午夜分，不见天日。”
  3. 分配；分享。《曹刿论战》：“衣食所安，弗敢专也，必以分人。”
  4. 区分；辨别。《荷蓧老人》：“四体不勤，五谷不分。”
  5. 春分；秋分。《左传》：“日过分而未至。”
  6. 成；事物的在十分之一。《祭妹文》：“减一分则喜。”
  7. 长度单位，十分为一寸。《核舟记》：“舟首尾长约八分有奇。”

    fèn

  8. 职分；名分；本分。《屈原列传》：“怀王以不知忠臣之分，故内欺于郑袖。”?﹤指南录﹥后序》：“予分当引决。”
  9. 情分；情谊。曹植《赠白马王彪》：“恩爱苟不亏，在远分日亲。”
  10. 缘分；福分。刘禹锡《寄乐天》：“幸免于斯分非分。”
  11. 才分；天分。《颜氏家和训·杂艺》：“良由无分故也。”
  12. 料想。《苏武》：“自分已死久矣。”

闻：

  1. 听见；听到。《木兰诗》：“不闻机杼声，惟闻闻女叹息。”
  2. 听说；得知。《殽之战》：“寡君闻吾子将步师出于敝邑，敢犒从者。”
  3. 报告；上报。《童区寄传》：“愿以闻于官。”
  4. 见闻；知识。《屈原列传》：“博闻强志，明于治乱。”
  5. 名声；名誉。《论语》：“四十、五十而无闻焉，斯亦不足畏也已。”
  6. 闻名；传扬。《出师表》：“不求闻达于诸侯。”

报：

  1. 断狱,判决罪人。《韩非子·五蠹》：“闻死刑之报,君为流涕。”
  2. 报恩;报答。《陈情表》：“尽节于陛下之日长,报养刘之日短也。”
  3. 报仇;报怨。《荆轲刺秦王》：“然则将军之仇报,而燕国见陵之耻除矣。”
  4. 答复;回信。《廉颇蔺相如列传》：“计未定,求人可使报秦者。”《西门豹治邺》：“烦大巫妪为入报河伯。”
  5. 到……去,前往。《孔雀东南飞》：“卿但暂还家,吾今且报府。”

次：

  1. 临时驻扎和住宿。《殽之战》：“秦伯素服郊次。”《陈涉世家》：“又间令吴广之次所旁丛祠中。”《赤壁之战》：“操军不利，引次江北。”
  2. 按顺序排列，次序。《荆轲刺秦王》：“荆轲奉樊於期头函，而秦武阳奉地图匣，以次进。”《陈涉世家》：“陈胜吴广皆次当行。”
  3. 在排列上次一等。《<论语>十则》：“多见而识广，知之次也。”《谋攻》：“凡用兵之法……全军为上，破军次之。”
  4. 及，到。《刘东堂言》：“有指摘其诗文一字者，衔之次骨，或至相殴。”
  5. 次，回。表示动作的次数。《游黄山记》：“历险数次。”《失街亭》：“某引兵冲杀十余次。”
